CastMatch - Talent Dashboard Wireframes

Purpose: Central hub for talent users to manage their profile, track applications, view analytics, and access quick actions.

Information Architecture

  • Route: /dashboard (home)
  • User Type: Authenticated Talent
  • Priority: P1 - Critical Path
  • Dependencies: Authentication, Profile Service, Analytics API

Component Hierarchy

  • Dashboard Layout Container
    • Sidebar Navigation (persistent)
    • Main Content Area
      • Header Bar with Search & Notifications
      • Stats Overview Cards
      • Activity Feed
      • Quick Actions Grid
      • Profile Completion Widget
      • Notifications Panel

Interaction Specifications

  • Stats Cards: Click to drill down into detailed analytics
  • Activity Feed: Real-time updates with 5-second polling
  • Quick Actions: One-click access to frequent tasks
  • Notifications: Badge count updates, mark as read on click
  • Profile Widget: Visual progress indicator with task checklist
Desktop - Dashboard Home (1440x900) Default State
