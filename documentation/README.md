# Documentation

This directory contains all project documentation for the Casting AI platform.

## Structure

- **`architecture/`** - System architecture documentation
- **`api/`** - API documentation and schemas
- **`deployment/`** - Deployment guides and instructions

## Key Documents

- **`CLAUDE.md`** - Claude AI development documentation
- **`USER_STORIES.md`** - User stories and requirements
- **`IMPLEMENTATION_ROADMAP.md`** - Development roadmap
- **`PRODUCT_INFRASTRUCTURE_STORIES.md`** - Infrastructure requirements
- **`WORK_PLAN.md`** - Project work plan
- **`DETAILED_TODO_LIST.md`** - Detailed task breakdown

## Navigation

All documentation is cross-linked and can be browsed starting from the main [README.md](../README.md) at the repository root.

For architectural decisions and technical specifications, start with the `architecture/` directory.
