🌐 Network Error Patterns

📶
Connection Timeout
Auto-retry with exponential backoff. Show progress indicator and allow manual retry.

Mobile Optimizations:

  • 48px minimum touch targets for retry buttons
  • Progress animation to indicate active retry
  • Offline mode toggle for cached content
  • Network status detection and display
⚡
Slow Connection
Detect slow networks and enable data-saver mode automatically.

Mobile Optimizations:

  • Reduce image quality on slow connections
  • Disable auto-play videos
  • Show data usage estimates
  • Offer "lite" version of pages

📝 Form Error Patterns

⚠️
Validation Errors
Inline validation with clear error messages and correction suggestions.

Mobile Optimizations:

  • Error messages appear above keyboard
  • Automatic scroll to first error field
  • Haptic feedback on validation errors
  • Voice-over friendly error descriptions
📎
File Upload Errors
Progressive upload with resumable transfers and clear file requirements.

Mobile Optimizations:

  • Camera integration for direct photo capture
  • File compression before upload
  • Upload progress with pause/resume
  • Alternative upload methods (email, cloud)

🔐 Auth Error Patterns

🔑
Session Expired
Preserve user state and redirect to re-authentication with context.

Mobile Optimizations:

  • Biometric re-authentication when available
  • Auto-save form data before redirect
  • Quick login with saved credentials
  • Social login options for faster access
🚫
Access Denied
Clear explanation of permissions with upgrade or contact options.

Mobile Optimizations:

  • One-tap upgrade or contact buttons
  • Alternative feature suggestions
  • Account management shortcuts
  • Help center integration

🔄 Mobile Recovery Flow Patterns

Touch-optimized recovery workflows that guide users back to success

🎯
Smart Retry
Intelligent retry mechanisms with progressive delays and alternative options.

Implementation Details:

  • Exponential backoff: 1s, 3s, 9s delays
  • Maximum 3 automatic retries
  • Visual countdown for next retry
  • Option to retry immediately or wait
📱
Graceful Degradation
Simplified functionality during errors to keep users productive.

Implementation Details:

  • Offline mode with cached content
  • Essential features only during errors
  • Progressive enhancement when restored
  • Clear indication of limited functionality
🆘
Help Integration
Contextual help and support options directly within error states.

Implementation Details:

  • Live chat widget in error screens
  • Contextual FAQ suggestions
  • Screen sharing for support calls
  • Error report with device diagnostics
🔍
Alternative Paths
Multiple ways to complete tasks when primary path fails.

Implementation Details:

  • Suggest similar features or workarounds
  • Direct links to alternative workflows
  • Quick access to popular destinations
  • Breadcrumb navigation for context
💾
State Preservation
Automatically save user progress and restore context after errors.

Implementation Details:

  • Auto-save form data every 30 seconds
  • Restore user's place after reconnection
  • Maintain search filters and sorting
  • Remember user preferences during session
📊
Error Analytics
Track error patterns and user recovery success rates.

Implementation Details:

  • Error frequency and type tracking
  • Recovery path success metrics
  • User satisfaction after error resolution
  • A/B testing of error message variations
